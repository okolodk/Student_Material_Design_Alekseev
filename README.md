# Лабораторная работа №9–10: Список студентов

> Приложение для отображения списка студентов с использованием современных принципов Material Design 3 в Jetpack Compose.

---

## 1. Название проекта
**Student_Material_Design**  
Приложение демонстрирует реализацию списка студентов с поддержкой светлой и тёмной тем, кастомной цветовой схемы, типографики, форм (скруглений), анимаций и верхней панели (Top App Bar). Каждая карточка студента содержит аватарку, имя, возраст и раскрывающееся описание — всё оформлено в едином стиле Material Design 3.

---

## 2. Цель работы
Освоить применение принципов Material Design в Android-приложениях с использованием Jetpack Compose. В ходе лабораторной работы были изучены и реализованы:
- настройка кастомной цветовой схемы (светлая/тёмная тема);
- использование Material Theming (цвета, формы, типографика);
- создание адаптивных компонентов (Card, TopAppBar, IconButton);
- управление состоянием и анимация раскрытия контента;
- интеграция пользовательских шрифтов (Abril Fatface, Montserrat);
- работа с ресурсами (строки, изображения, размеры, формы).

---

## 3. Функциональность
Приложение реализует следующие возможности:
- Отображение списка из 10 студентов с аватарками (`student1.jpg`–`student10.jpg`);
- Карточки студентов с именем (жирный Montserrat, 20sp), возрастом (обычный Montserrat, 14sp) и круглой аватаркой;
- Возможность раскрытия/сворачивания подробного описания студента по нажатию на кнопку ▼/▲;
- Плавная анимация изменения высоты карточки при раскрытии (с пружинной анимацией);
- Поддержка светлой и тёмной тем (автоматически подстраивается под системные настройки);
- Верхняя панель (Top App Bar) с логотипом и названием приложения в стиле `displayLarge` (Abril Fatface, 36sp);
- Адаптивное оформление через Material Theme: цвета, формы (скруглённые углы карточек), отступы и шрифты.

---

## 4. Технологии
Для разработки использовались:
- **Jetpack Compose** — декларативный UI-фреймворк;
- **Material Design 3** — система дизайна (цвета, типографика, формы, компоненты);
- **Kotlin** — язык программирования;
- **Android SDK API 25+** — минимальная версия целевой платформы;
- **Google Fonts**: Abril Fatface (заголовки), Montserrat (основной текст);
- **Ressource system**: `strings.xml`, `dimens.xml`, `drawable/`, `font/`;
- **Compose Navigation** — не используется (одноэкранный интерфейс);
- **Gradle** — сборка и управление зависимостями (включая `material-icons-extended` для иконок).
